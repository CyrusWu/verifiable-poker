# ♠ 可验证德州扑克 · Verifiable Texas Hold'em

一个**单文件、零后端、可证公平**的德州扑克练习桌。打开网页就能玩,和 6 种风格迥异的 AI 对手过招,还有一个只讲数学、不讲客套的实时教练。

**▶ 直接玩: https://cyruswu.github.io/verifiable-poker/**

## 特点

- **可证公平 (provably fair)**:每手发牌用 commit-reveal——发牌前先公示 `SHA256(serverSeed)`,你的 client seed 参与洗牌,牌局结束揭示 seed,一键 Verify 可逐字节复算整副牌序。庄家(代码)想换牌,数学上会被抓。
- **六个 AI 对手 × 五档难度**:紧凶冠军、松被动土豪、GTO 数学家、疯狂网红、岩石银行家、谨慎跟注型——风格参数(松紧/激进/诈唬/黏性)全部可调。难度 **L1 鱼 → L5 宗师**:等级越高,范围越准、越会用位置施压、越难被诈唬。
- **天梯(诚实榜)**:挑战不同等级 AI 拿分——`分 = Σ(每手净额 ÷ 大盲 × 等级系数)`,L1 ×0.2 → L5 ×3.5,**虐鱼几乎不加分,赢宗师才值钱**。纯荣誉、不涉钱。
- **教练模式**:直播间式实时点评。你每出一手,教练即时告诉你打得好不好、为什么——判据是职业玩家的速算工具箱:决策时胜率 vs 底池赔率、**按位置的翻前开牌范围(RFI)**、**短码 push/fold 表**、**outs ×4/×2 速算**、**隐含赔率**、**SPR 承诺度**;每手结束给要点小结;还能诊断"上头"(大输后变松变凶)。
- **牌局记录**:每手完整记录 + 防篡改哈希链;盈亏曲线、VPIP/PFR/AF/WTSD 数据面板。
- **责任博弈**:损失上限、休息提醒、破产须手动确认补码。

## 诚实声明

教练是**漏洞检测器,不是盈利保证**。范围表 + 启发式(outs 速算/隐含赔率/SPR)是职业玩家的**速算近似**,蒙特卡洛胜率是估算——加在一起也**不是** GTO solver(没有 CFR 均衡计算);教练绿灯 ≠ +EV。真实利润永远来自对手犯错,不来自任何公式。

**天梯是诚实榜**:分数由玩家浏览器本地算完上传,服务端**不校验真伪**——朋友之间纯荣誉、不涉钱,靠自觉。身份=浏览器本地 ID,清缓存即丢失。

当前为**单人 vs AI**(每位玩家各开一桌);真人同桌联机在路线图上。仅供练习娱乐,无真钱,请勿用于赌博。

## 技术

单个 `index.html`,无依赖、无构建、离线可玩(`file://` 直接打开也行)。SHA-256、Fisher–Yates 确定性洗牌、多人边池/不足额加注规则、蒙特卡洛胜率引擎全部内联实现。

天梯服务在 `server/ladder/`(Cloudflare Workers + D1,免费档,4 步部署见其 README)。**不部署也能玩**——天梯显示"未开通",分数照样本地累计。
