## 它解决什么

新手学德扑最大的问题是**没有反馈**：你打完一手，赢了输了都不知道自己打得对不对。这个教练把反馈做到实时——

- **决策前**给你事实：你在什么位置（"按钮位 · 最后行动 · 最有利"，不用背"枪口位"）、你的牌是什么概念（"顶对"/"同花听牌 9 张"）、实时胜率、跟注要多少赔率。
- **决策后**判你对错：用职业玩家的速算工具箱——决策时胜率 vs 底池赔率、**按位置的翻前开牌范围(RFI)**、**短码 push/fold 表**、**outs ×4/×2 速算**、**隐含赔率**、**SPR 承诺度**。
- **课程化**：一次只盯你一个漏洞（一次改一件事 > 一次报五条），教练还能自动挑你最大的复发漏洞当本期课题。
- **行为诊断**：包括"上头"检测——大输之后你是不是变松变凶了。

教练有三种风格随你换：**严师**（"错。"）/ **鼓励**（"这步能更好——"）/ **数据**（"[-EV]"）。

## 其它

- **五档 AI**：L1 鱼 → L5 宗师。等级越高，范围越准、越会用位置施压、越难被诈唬。**不是作弊看牌**。
- **天梯**：每个难度一张榜，口径 = **BB/100**（每百手赢几个大盲，扑克标准胜率），满 30 手上榜。**天梯模式 = 排位赛**：选难度 + 填昵称后每手自动计入；休闲局不计分。纯荣誉，不涉钱。
- **经典复盘**：重放扑克史上的名局（Moneymaker 的世纪诈唬、Ivey vs Jackson 的诈唬战），到关键决策点**先问你怎么打**，再揭示高手的选择和理由。
- **可验证发牌**：每手用 commit-reveal——发牌前公示 `SHA256(serverSeed)`，你的 client seed 参与洗牌，牌局结束揭示 seed，一键 Verify 可逐字节复算整副牌序。庄家想换牌，数学上会被抓。
- **牌局记录**：每手完整记录 + 防篡改哈希链；盈亏曲线、VPIP/PFR/AF/WTSD。
- **责任博弈**：损失上限、休息提醒、破产须手动确认补码。

## 诚实声明

教练是**漏洞检测器，不是盈利保证**。范围表 + 启发式是职业玩家的**速算近似**，蒙特卡洛胜率是估算——加在一起也**不是** GTO solver（没有 CFR 均衡计算）；教练绿灯 ≠ +EV。真实利润永远来自对手犯错，不来自任何公式。

**天梯是诚实榜**：成绩由玩家浏览器本地算完上传，服务端**不校验真伪**——朋友之间纯荣誉、靠自觉。身份 = 浏览器本地 ID，清缓存即丢失。

**经典复盘的内容都逐项核过公开资料**，来源与核实状态写在每手开场页；核不实的细节宁可不写、或从可信处起放——不编造史料。

当前为**单人 vs AI**（每位玩家各开一桌）；真人同桌联机在路线图上。仅供练习娱乐，无真钱，请勿用于赌博。

## 技术

单个 `index.html`，无依赖、无构建、离线可玩（`file://` 直接打开也行）。SHA-256、Fisher–Yates 确定性洗牌、多人边池/不足额加注规则、蒙特卡洛胜率引擎全部内联实现。

天梯服务在 `server/ladder/`（Cloudflare Workers + D1，免费档，4 步部署见其 README）。**不部署也能玩**——天梯显示"未开通"，成绩照样本地累计，开通后自动同步。
